### W1 — POST /api/projects path-traversal (SECURITY-WARN)
`projects.name` accepted any `min_length=1` string and piped to `Path(repo_root) / 'context' / 'projects' / project_name` at `services/project_scaffold.py:85`. A name like `'../foo'` resolves through `Path.mkdir(parents=True)` and creates dirs + zero-byte `.gitkeep` files outside the scaffold root. Bounded blast radius (mkdir + touch + template copyfile only — no content overwrite due to `if dest.exists(): continue`), but a real write primitive that should not survive into Phase 4 (auth landing).

**Filed as: Kanban #121 (SECURITY-WARN → priority HIGH, dev-backend).** Fix shape: Pydantic charset regex `^[a-zA-Z0-9_-]{1,64}$` + independent traversal guard inside `scaffold_project_folder` (defense-in-depth).

### W2 — POST /api/tasks 400 detail leaks raw PG (SECURITY-WARN)
`POST /api/tasks` returns `str(exc.orig)` raw on `IntegrityError` — exposes asyncpg exception class name + raw `DETAIL:` line + constraint name. PATCH path mirrors M5 hygiene; POST regresses.

**Filed as: Kanban #122 (SECURITY-WARN → priority HIGH, dev-backend).** Fix shape: mirror PATCH update_task IntegrityError translation block; pin via source-text test.

### N1 — APP_DEBUG=True default (SECURITY-NIT)
Production-sensitive flag defaults to True; should fail-CLOSED.

### N2 — `_DEFAULT_REPO_ROOT` resolution brittleness (SECURITY-NIT)
`Path(__file__).resolve().parent.parent.parent` is brittle to source-tree refactors. Prefer required `REPO_ROOT` env var (fail-fast on unset).

**Both NITs filed as part of Kanban #123 (SECURITY-NIT bundle, priority LOW, dev-backend).**
